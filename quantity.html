Componente

Quantity stepper

Selector numérico com botões − e + para alterar quantidades. Usado no carrinho, PDP (configurações de quantidade) e em formulários onde se precisa de incremento controlado.

<div class="qty">
  <button class="qty__btn" type="button" aria-label="Diminuir">−</button>
  <input class="qty__input" type="number"
         value="1" min="1" max="99"
         aria-label="Quantidade" />
  <button class="qty__btn" type="button" aria-label="Aumentar">+</button>
</div>
Sizes

Sizes

Três tamanhos. md default em PDP, sm em mini-carrinho e linhas de carrinho compactas, lg em cart full-page.

<div class="qty qty--sm">...</div>
<div class="qty">...</div>          <!-- default -->
<div class="qty qty--lg">...</div>
States

States

Mínimo atingido (− disabled)

Quando o valor está no mínimo, o botão − fica disabled.

<button class="qty__btn" type="button" disabled
        aria-label="Diminuir">−</button>

Máximo atingido (+ disabled)

Quando o valor é igual ao stock disponível, + disabled.

<input class="qty__input" type="number" value="5" max="5" />
<button class="qty__btn" type="button" disabled
        aria-label="Aumentar">+</button>

Usage

Do's & Don'ts

✓ Faz

Disable − no min e + no max. Feedback visual imediato.

✗ Evita

Permitir clicar e mostrar erro depois. Frustra utilizador.

✓ Faz

Permitir input direto no campo (digitação). Para utilizadores que sabem a quantidade exata.

✗ Evita

Bloquear input — só botões. 50× clique no + para chegar a 50 é frustrante.

✓ Faz

Validar input numérico e respeitar min/max em onChange.

✗ Evita

Aceitar texto, decimais ou números negativos. Quantidade é inteiro positivo.

Acessibilidade

Acessibilidade

Aria-labelBotões: "Diminuir quantidade" e "Aumentar quantidade". Input: "Quantidade".
Aria-liveConsiderar aria-live="polite" num span anuncio fora do campo: "Quantidade alterada para 3".
Typetype="number" com inputmode="numeric" (mobile abre teclado numérico).
Min/maxAtributos HTML respeitados pelo browser. Botões disabled refletem visualmente.
Touch targetMínimo 36px (default), 44px (lg) para mobile.
