Usage

Do's & Don'ts

✓ Faz

"99+" para números grandes. Após 99, mostrar "99+" — manter o badge compacto.

✗ Evita

"143" ou "1247" no badge. Estraga o layout e o utilizador raramente precisa do número exato.

✓ Faz

Esconder o badge quando o número é 0. Ausência de badge = ausência de items.

✗ Evita

Mostrar "0" — ruído visual sem valor informativo.

✓ Faz

Dot quando não interessa o número exato — só "tens algo novo".

✗ Evita

Misturar Badge (numérico) com Sticker (estado) — são componentes diferentes.

Acessibilidade

Acessibilidade

Screen readerBadge tem que ser anunciado em contexto. Adicionar aria-label no wrapper: aria-label="Carrinho com 3 items".
Dot indicatorDot sozinho é invisível para screen reader — adicionar aria-label="3 notificações novas".
Live regionQuando o número muda dinamicamente, considerar aria-live="polite" para anunciar.
ContrasteCor sobre fundo escuro/claro tem contraste > 4.5:1. Texto sempre branco sobre cor.
