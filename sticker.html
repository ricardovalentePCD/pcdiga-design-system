Sticker

Etiquetas pequenas para comunicar estado de stock, promoções, novidades ou descontos. São o sinal visual mais rápido na PCDIGA.

Usage

Do's & Don'ts

✓ Faz

Usar a variante semântica certa: available para stock, discount só para % numéricas.

✗ Evita

"Disponível" em sticker amarelo (discount). O utilizador associa cada cor a uma semântica.

✓ Faz

Máximo 2 stickers por card. Mais que isso é ruído.

✗ Evita

5 stickers num único card. O utilizador não consegue priorizar a informação.

✓ Faz

Texto curto e directo: "Pré-venda", "-30%", "Novo". Uma palavra é suficiente.

✗ Evita

Frases inteiras dentro de stickers: "Disponível em loja Lisboa". Sticker é etiqueta, não label.

Acessibilidade

Acessibilidade

Cor não bastaCada sticker tem texto explícito. Daltónicos identificam pelo texto, não só pela cor.
ContrasteTodas as combinações background+texto passam WCAG AA (verificadas no Figma).
Screen readerTexto interno é lido normalmente. Não usar aria-hidden.
Tamanho mínimoMesmo em sm mantém 11px — não descer abaixo disso.
